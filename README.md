# The Act

The Act is a modern and fully responsive skincare & body care landing page built using HTML, CSS, and JavaScript. The website includes a hero section with brand messaging, a navigation menu for product categories, a philosophy/about section, a new products showcase, and a footer with newsletter signup and customer care links. The design features a clean, minimal aesthetic with elegant serif typography and a soft, editorial color palette.

## Tech Stack

- HTML5
- CSS3
- JavaScript

## Features

- Responsive design for all screen sizes
- Clean, minimal editorial layout
- Category navigation (New, Best Sellers, Skincare, Hair & Body, Kids, Accessories)
- Product showcase sections with imagery
- Newsletter signup section
- Footer with company, customer care, and gift card links
- Mobile-friendly navigation
